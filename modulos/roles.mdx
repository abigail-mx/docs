<Warning>
Para visualizar y gestionar los roles de cada usuario debe tener acceso al módulo de **Roles**.
</Warning>

## Navegación y acceso

**Accede al módulo de Roles**
Como administrador, en la barra de navegación busca y selecciona el apartado **Administrativo** y después haz clic en **Roles**

Una vez estando en dicho módulo, podrá observar el rol llamado "Admin" que es rol por defecto que otorga abigail.
<Note>
Para crear roles adiciones comprueba las posibilidades de su [**Membresia**](https://www.abigail.mx/planes)
</Note>

Para administrar los roles de cada usuario basta con hacer clic en el icono de (>) gris de la parte derecha.

En el caso del rol **Admin** después de haber ingresado, podrá ver dos apartados: 

## Lista de Usuarios Activos con Permisos de Administrador
   - Muestra todos los usuarios activos con permisos de administrador.
   - Puede buscar por nombre, apellido y correo electrónico.
   - Puede dar de baja usuarios haciendo clic en el icono de (X) rojo.

##  Información del Rol
    - Muestra el nombre del Rol.
    - Muestra el catalogo de modulos al que el los usuarios con ese rol tienen permisos de acceso y a los que no.

<Warning>
En caso de haber realizado un cambio es fundamental haber pulsado el boton Guardar cambios de la parte final.
</Warning>

## Creación de un Nuevo Rol
Para crear un nuevo rol es tan fácil como precionar el boton **Crear Rol** dentro de este mismo modulo. 

Después haber presionado el boton, lo primero que se le solicitará será establecer un nombre para el nuevo rol. Posteriormente, unicamente tendrá que elegir los módulos que ese nuevo rol tendrá permitido acceder y listo.
Basta con presionar el boton **Crear rol** en la parte de abajo para guardar los cambios y el nuevo rol aparecerá en la página principal del modulo.

<Note>
Recuerde que para crear roles adicionales debe tener un plan [**Básico**](https://www.abigail.mx/planes) o un plan [**Premium**](https://www.abigail.mx/planes).
</Note>
